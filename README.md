# 🚀 Express Clean Architecture API

A scalable Node.js backend built with Express, following clean architecture principles.  
This project demonstrates DTO-based validation using Joi, centralized error handling, and a modular folder structure.

---

## 📌 Features

- ✅ Modular architecture (feature-based structure)
- ✅ DTO-based validation using Joi
- ✅ Reusable BaseDTO pattern
- ✅ Centralized error handling (`ApiError`)
- ✅ Standardized API responses (`ApiResponse`)
- ✅ Middleware-based validation
- ✅ MongoDB integration using Mongoose
- ✅ Authentication-ready structure (register/login flow)
